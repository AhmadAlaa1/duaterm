Prayer in your Terminal for Quran, Azkar, and the 99 Names of Allah

Download GitHub Install Preview

Install with one command

curl -fsSL https://raw.githubusercontent.com/AhmadAlaa1/noorterm/main/install.sh | NOORTERM_GITHUB_REPO=AhmadAlaa1/noorterm bash

The installer detects Debian/Ubuntu or Fedora/RHEL automatically and installs the latest release package with the native package manager.

What the product includes

  • Offline Quran reader with bundled data
  • Morning Azkar and Night Azkar
  • 99 Names of Allah in Arabic
  • Web UI
  • Native Linux packages for Debian, Fedora, and Arch(Coming Soon!)
